EL ZORZAL

Desarrollo residencial moderno

Proyecto

Proyecto de desarrollo residencial moderno localizado en Santa Lucía, San Juan, Argentina. Con una estética cálida y contemporánea, el conjunto propone unidades funcionales con espacios luminosos, ventilación cruzada y una relación directa con el espacio exterior.

Conjunto

El proyecto se implanta sobre un terreno de configuración triangular, cuya particular geometría dio origen a una propuesta arquitectónica cuidadosamente estudiada. A partir de esta condición se priorizó la orientación de las unidades residenciales con el objetivo de favorecer la ventilación cruzada y optimizar el ingreso de luz natural.
